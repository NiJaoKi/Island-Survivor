# Island Survivor (текстовая браузерная RPG)

Игра-прототип в духе “SAO”: создаёшь персонажа **без класса**, выбираешь **одно оружие ближнего боя**, качаешь **уровень** и **навык оружия**, фармишь **Коллы** и лут с мобов.  
**Перманентная смерть:** умер — персонаж удаляется полностью.

## Запуск

1. Открой файл `play.html` в браузере (Chrome/Edge/Firefox) — это “вход/ссылка” в игру.
2. Нажми “Войти в игру”.
  
Либо напрямую открой `index.html`.
3. Играй.

Прогресс хранится в `localStorage` браузера. При смерти — автоматически стирается.

## Валюта

Деньги отображаются как **Коллы**:
- `1 колл`, `10 колл`, `100 колл`, `1000 колл`, `10000 колл` …
- при больших суммах: `1,23 МКолл` (мегаколл)

## Файлы

- `index.html` — интерфейс
- `style.css` — стили
- `game.js` — логика (персонаж/мобы/бой/лут/перманентная смерть)
